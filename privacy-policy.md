# Flickr Metadata Assistant Privacy Policy

Publisher: KodeXpo.

This extension processes Flickr page data so it can generate, review, and apply photo metadata only for the user's own Flickr photos and other content they own or control. Depending on the selected provider, photo text, titles, descriptions, tags, and image data may be sent to a third-party AI provider such as OpenAI, DeepSeek, or a local Ollama instance to generate draft metadata.

Data handled by the extension includes authentication information such as Flickr OAuth tokens and account identifiers, website content such as Flickr page titles, descriptions, tags, and image data, and locally stored draft metadata and settings. The extension stores this information locally in Chrome on the user's device. It does not sell user data or use it for advertising. Any data sent to third-party AI providers is used only to provide the extension's metadata drafting feature.

The extension only processes the user's own Flickr content and respects Flickr photo visibility and owner restrictions. Users are responsible for ensuring that any metadata changes and publication actions are appropriate for their photos.

This extension is a drafting and publishing aid. It does not guarantee the accuracy, legality, or suitability of generated metadata. You are responsible for reviewing all titles, descriptions, tags, visibility changes, and publication actions before applying them.

This extension is provided "as is" and "as available." We make no warranty that generated or edited metadata is accurate, complete, suitable, or legally safe. You are solely responsible for reviewing all titles, descriptions, tags, visibility changes, and publication actions before applying them. Use this extension only for photos and content you own or control. We are not responsible for errors, data loss, or issues caused by Flickr or third-party AI providers.

This product uses the Flickr API but is not endorsed or certified by SmugMug, Inc.
