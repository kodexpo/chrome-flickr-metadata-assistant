# Flickr Metadata Assistant

Standalone Chrome extension for drafting Flickr photo metadata directly in the browser.
Published by KodeXpo.

## What it does

- Reads the current Flickr photo page
- Generates title, description, and tags with a local or cloud AI provider
- Lets you edit the draft in the popup
- Applies the draft back to the Flickr page fields

## Supported providers

- `local` for Ollama vision models
- `openai` for OpenAI vision models
- `deepseek` for text-only draft cleanup
- `heuristic` for offline fallback
- `auto` to try the configured providers in order

## Install

1. Open `chrome://extensions`
2. Enable Developer mode
3. Choose `Load unpacked`
4. Select the `chrome-extension/` directory

## Configure

Open the extension options page and enter any API keys or model names you want to use.

Settings are stored in `chrome.storage.local`.

If you want a reinstall-safe copy, use the Options page backup tools to export a JSON backup. It includes the stored Flickr credentials, AI settings, and saved draft state so you can restore them after uninstalling or clearing Chrome data.

If Flickr asks for a callback URL, use the one shown on the options page. It is generated from `chrome.identity.getRedirectURL('flickr')`.

The options page also includes an About / Legal section and a privacy policy link in `privacy-policy.html`.

The extension is intended for the user's own Flickr photos and other content they own or control.
It handles authentication information such as Flickr OAuth tokens and account identifiers, plus website content such as Flickr page titles, descriptions, tags, and image data, all stored locally in Chrome unless sent to an enabled AI provider for drafting.

The extension is provided as-is. Generated or edited metadata should always be reviewed before applying or publishing, and the user is responsible for ensuring their use complies with Flickr terms and any applicable rights or restrictions.

If you uninstall the extension, Chrome removes its local storage. Export the backup JSON first if you want to keep your credentials and drafts.

For Flickr auth, you will need:

- Flickr API key
- Flickr API secret
- access token generated by the extension after you connect

## Use

1. Open a Flickr photo page
2. Click the floating `Open Flickr Metadata Assistant Panel` button on the page, or the extension icon
3. Generate a draft
4. Edit the fields if needed
5. Apply the draft to the page or publish it directly to Flickr
